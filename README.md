# Conversor de Monedas 💱

Proyecto del Challenge ONE - Oracle Next Education.

## Descripción

Aplicación de consola en Java que permite convertir monedas utilizando una API externa.

## Tecnologías

Java 17  
Gson  
HTTP Client  
API ExchangeRate

## Funcionalidades

- Conversión USD → ARS
- Conversión ARS → USD
- Conversión USD → BRL
- Conversión BRL → USD
- Conversión USD → COP
- Conversión COP → USD

## Ejecución

Compilar:

mvn clean install

Ejecutar:

java -jar conversor-moneda.jar
